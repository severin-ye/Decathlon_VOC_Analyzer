# 상품 증거 모델

## 상품 증거 패키지의 구성

각 상품은 product_id 를 핵심으로 하여 다음 증거를 탑재합니다.

- 기본 정보: 제목, 카테고리, 가격, 브랜드, SKU
- 상품 문안: 짧은 설명, 긴 설명, 판매 포인트, 파라미터, 사양
- 상품 이미지: 메인 이미지, 디테일 이미지, 장면 이미지, 구조 이미지, 사이즈 이미지

## 핵심 원칙

상품 이미지와 상품 텍스트는 두 개의 독립 시스템이 아니라, 동일한 상품 증거 패키지의 두 가지 모달 관점입니다.

## 최소 필드 제안

### 상품 텍스트 블록

- text_block_id
- product_id
- text_type
- content
- source_section

### 상품 이미지

- image_id
- product_id
- image_type
- image_path 또는 image_url
- optional aux_text

## aux_text 의 역할

Native Multimodal 경로를 사용하더라도, 상품 이미지에 대해 가벼운 보조 텍스트를 유지할 것을 권장합니다.

- 이미지 제목
- 짧은 설명
- 페이지 위치
- 선택적 OCR 요약

이것은 원시 모달을 대체하기 위한 것이 아니라, 이후의 디버깅, 필터링, 표시, 오류 분석을 더 쉽게 하기 위한 것입니다.
