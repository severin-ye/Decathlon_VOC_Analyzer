# 용어집

## 핵심 용어

- 멀티모달 RAG: 텍스트, 이미지, 표, 차트, 오디오, 비디오 등의 내용을 함께 검색과 생성 과정에 포함시키는 것.
- 원시 모달: 이미지, 표 스크린샷, 페이지 스크린샷 등의 원본 내용을 가능한 한 유지하고, 먼저 전부 순수 텍스트로 바꾸지 않는 것.
- Native Multimodal: 검색 시 멀티모달 임베딩을 사용하고, 생성 시 직접 멀티모달 대형 모델을 사용하는 것.
- 증거 기반: 모든 중간 결론과 최종 결론은 반드시 명확한 증거로 되돌아갈 수 있어야 한다.
- 설명 가능성: 각 결론이 어떤 리트리벌 결과, 리뷰 조각, 통계 결과, 모델 출력에서 왔는지 추적할 수 있는 것.

## 리트리벌 관련 용어

- Page-level retrieval: 전체 페이지 이미지 또는 전체 페이지 객체를 리트리벌 단위로 삼는 것.
- Region-level retrieval: 페이지 내부의 국소 영역을 리트리벌 단위로 삼는 것. 예를 들면 차트, 표, 국소 구조도.
- Multi-vector retrieval: 하나의 문서 객체가 하나의 벡터가 아니라 여러 개의 벡터에 대응하는 것.
- Late interaction: 질의 벡터와 문서의 여러 국소 벡터를 각각 매칭한 뒤 점수를 집계하는 것.
- Reranker: 초기 리트리벌 결과에 대해 두 번째 단계 재정렬을 수행하는 것.

## 프로젝트 도메인 용어

- 상품 증거: 상품 이미지, 상품 제목, 핵심 판매 포인트, 파라미터, 사양 등 상품 자체와 관련된 내용.
- 리뷰 증거: 리뷰 텍스트에서 인용 가능한 의견, 감정, 사용 장면, 원문 구절.
- Aspect: 리뷰의 측면 또는 차원. 예를 들어 쿠셔닝, 통기성, 편안함, 외관.
- Product impression: 특정 상품을 중심으로 집계하여 얻는 인상 모델.
- Customer impression: 특정 사용자 요구 또는 사용 장면을 중심으로 집계하여 얻는 고객 인상 모델.
