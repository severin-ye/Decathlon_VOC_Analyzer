# 상품 문안 리트리벌

## 리트리벌 대상

상품 문안 리트리벌은 다음을 포괄합니다.

- 제목
- 판매 포인트 bullet points
- 긴 설명
- 파라미터
- 사양 설명

## 권장 모델

리뷰 query 와 상품 텍스트가 동일한 의미 공간에 들어갈 수 있도록, 동일한 멀티모달 embedding 모델로 텍스트를 인코딩한다.

## 온라인 질의 로직

1. 리뷰 측면 객체에서 query 를 생성한다.
2. 상품 문안 인덱스에서 관련 text_block_id 를 리트리벌한다.
3. 텍스트 후보와 embedding score 를 반환한다.

## 이 계층이 해결하는 문제

- 리뷰 관점에 대해 상품 공식 설명 또는 설계 의도를 찾아준다.
- 이후 “사용자 인지 문제”와 “문안/구조에 이미 명확한 설명이 있지만 사용자가 인정하지 않는 경우”를 구분할 수 있도록 지원한다.

## 왜 이 계층에만 의존할 수 없는가

많은 상품 특성은 뚜렷한 시각 속성을 가지므로, 텍스트 설명만으로는 사용자의 인지를 충분히 설명할 수 없기 때문이다.
