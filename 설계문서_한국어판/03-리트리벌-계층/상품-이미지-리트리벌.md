# 상품 이미지 리트리벌

## 리트리벌 대상

이미지 리트리벌이 포괄하는 대상은 다음과 같습니다.

- 메인 이미지
- 디테일 이미지
- 장면 이미지
- 구조 이미지
- 이후 확장 가능한 국소 크롭 이미지

## 권장 모델

Qwen3-VL-Embedding 을 사용해 이미지의 벡터 표현을 구축한다.

## 온라인 질의 출처

질의는 보통 리뷰 측면 객체에서 오며, 예를 들면 다음과 같습니다.

- 쿠셔닝이 좋다
- 여름에 답답하다
- 외관이 예쁘다
- 수납 구획이 적다

## 온라인 질의 로직

1. 리뷰 측면 객체에서 간결한 query 를 생성한다.
2. query 를 벡터로 인코딩한다.
3. 상품 이미지 인덱스에서 유사도 검색을 수행한다.
4. 후보 image_id 와 그 점수를 반환한다.

## 이 계층이 해결하는 문제

- 리뷰의 관점이 상품 시각 증거에 대응할 수 있도록 한다.
- 이후 “구조 개선 제안”에 이미지 근거를 제공한다.

## 후속 확장

대화에서는 더 강한 방향이 제시되었습니다.

- page-level 에서 region-level 로의 진화
- 국소 크롭 이미지의 도입
- 더 세밀한 시각 증거 위치 지정
