# 상품 인상과 고객 인상

## 상품 인상

상품 인상 모델은 상품별로 리뷰 측면 객체를 집계한 뒤 얻는 구조이며, 다음에 주목한다.

- 높은 빈도의 장점
- 높은 빈도의 단점
- 논쟁 지점
- 장면화된 평가

대화에서 제시된 전형적 측면은 다음을 포함한다.

- 편안함
- 통기성
- 경량성
- 수납성
- 내구성
- 가성비
- 외관
- 사이즈 정확성

## 고객 인상

고객 인상 모델은 사용자의 실제 신원 프로필이 아니라, 리뷰 의미 속 수요 클러스터이며, 다음에 주목한다.

- 가벼운 하이킹 사용자
- 통근 사용자
- 여름철 사용 사용자
- 경량성을 중시하는 사용자
- 외모와 디자인을 중시하는 사용자
- 가성비를 중시하는 사용자

## 본 계층의 가치

- 서로 다른 상품이 서로 다른 사용 장면에서 어떻게 평가되는지 이해하는 데 도움을 준다.
- 서로 다른 고객 수요 집단이 동일 상품에서 무엇에 주목하는지 비교하는 데 도움을 준다.
- 이후 제안 생성에 인구 집단 맥락을 제공한다.

## 출력 형식

이 계층 출력의 핵심은 자유 텍스트 한 문단이 아니라, 일련의 집계 결과이다.

- aspect_frequency
- positive_ratio
- negative_ratio
- scene_distribution
- representative_reviews
