# 논문과 오픈소스 프로젝트

상세 인용 정리는 [인용-개요](인용/인용-개요.md) 를 참조한다. 이 페이지에는 빠르게 훑어보기 위한 고수준 분류 색인만 남긴다.

## 대화에서 다룬 주요 방향

### 원시 모달 리트리벌 주선

- ColPali
- ColQwen
- VisRAG

이 프로젝트들은 “page image 또는 region image 를 직접 유지하여 검색하는” 경로를 대표하며, multi-vector 와 late interaction 을 강조한다.

### 구조화 멀티모달 리트리벌 주선

- RAG-Anything
- NVIDIA Nemotron / NeMo Retriever pipeline

이 경로들은 구조화 요소 보존, 관계 모델링, 융합형 리트리벌을 더 강조한다.

### 문서 파싱 관련

- Docling

PDF, HTML 및 복잡한 문서 구조 파싱에 적합하지만, 현재 데카트론 HTML 상품 페이지 시나리오에서는 1순위가 아니라 선택적 확장에 속한다.

### 오케스트레이션과 모델

- LangChain
- ChatQwen
- Qwen3-VL-Embedding
- Qwen3-VL-Reranker

### 벡터 데이터베이스

- Qdrant

## OpenClaw 에 대한 결론

대화에서 형성된 결론은 다음과 같다. OpenClaw 와 ClawRAG 는 현재 이 프로젝트에 가장 적합한 원시 모달 리트리벌 주선이 아니므로, 여기서는 핵심 기술 의존성이 아니라 참고 배경으로만 남긴다.
