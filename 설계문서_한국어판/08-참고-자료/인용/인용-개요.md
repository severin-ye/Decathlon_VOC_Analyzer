# 인용 디렉터리 총람

이 디렉터리는 원본 대화에서 등장한 외부 링크 자료를 축적하여, 이후 논문 작성, 방안 발표, 방법론 장 작성에 사용할 수 있도록 하기 위한 것입니다.

## 정리 원칙

각 인용은 가능한 한 통일된 구조로 기록한다.

- 외부 링크
- 대응 내용 정리
- 본 프로젝트에서 사용하는 부분
- 사용하는 이유
- 본 프로젝트에서 사용하는 아이디어

## 사용 제안

이후 논문을 쓴다면:

- 방법 배경에는 기본 개요, 원시 모달 리트리벌, 구조화 리트리벌 세 그룹 자료를 우선 인용할 수 있다.
- 시스템 구현 부분에는 LangChain, Qwen, Qdrant, Docling 을 우선 인용할 수 있다.
- 관련 작업 비교 부분에는 “후보였지만 채택하지 않은 경로”의 OpenClaw, ClawRAG 등의 자료를 직접 사용할 수 있다.

## 포함하지 않은 외부 링크

다음 외부 링크는 정식 인용 디렉터리에 포함하지 않았다.

- 대화 자체의 ChatGPT 공유 링크
- ChatGPT Exporter 페이지

이유는 이들이 프로젝트의 기술 근거가 아니며, 논문 인용 출처로도 적합하지 않기 때문이다.
