# 설계 문서 개요

이 디렉터리는 [대화 기록 원문](../gpt聊天记录.md) 의 프로젝트 설계 논의를 유지보수 가능하고, 확장 가능하며, 단일 책임을 갖는 문서 시스템으로 정리하기 위해 사용됩니다.

## 문서 범위

- 이 디렉터리는 시스템 설계, 기술 경로, 데이터 모델링, 리트리벌 설계, 생성 로직, 설명 가능성 및 구현 순서에 초점을 둡니다.
- 이 디렉터리는 원본 대화 전문을 저장하지 않으며, 원본 출처는 여전히 [대화 기록 원문](../gpt聊天记录.md) 를 기준으로 합니다.
- 각 문서는 하나의 주제만 담당하며, 배경, 구현, 선택안, 평가를 하나의 파일에 섞지 않습니다.
