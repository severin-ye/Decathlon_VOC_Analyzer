# 论文与开源项目

详细引用整理见 [引用/README.md](%E5%BC%95%E7%94%A8/README.md)。本页保留的是高层分类索引，供快速浏览使用。

## 对话中涉及的主要方向

### 原始模态召回主线

- ColPali
- ColQwen
- VisRAG

这些项目代表的是“直接保留 page image 或 region image 做检索”的路线，强调 multi-vector 与 late interaction。

### 结构化多模态召回主线

- RAG-Anything
- NVIDIA Nemotron / NeMo Retriever pipeline

这些路线更强调结构化元素保留、关系建模和融合式召回。

### 文档解析相关

- Docling

适用于 PDF、HTML 和复杂文档结构解析，但在当前迪卡侬 HTML 商品页场景中属于可选扩展，而不是第一优先级。

### 编排与模型

- LangChain
- ChatQwen
- Qwen3-VL-Embedding
- Qwen3-VL-Reranker

### 向量库

- Qdrant

## 对 OpenClaw 的结论

对话中形成的结论是：OpenClaw 及 ClawRAG 当前并不是本项目最合适的原始模态召回主线，因此这里只保留为参考背景，而非核心技术依赖。
