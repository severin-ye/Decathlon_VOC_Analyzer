
Title

QAGS Revisited / QA-based Factual Consistency for Summarization

Year

2024（新版扩展）

Venue

ACL / arXiv

Task

summary factual consistency

Core Idea

把 summary 转换成 QA 任务：

👉 如果 QA answer 不一致 → summary 是 hallucinated

Framework Type

✅ QA-based

Pipeline
Step 1: Question Generation

从 summary 生成问题：

Summary: John won the race in 2020.

→ Q: When did John win the race?
Step 2: Dual QA
用 summary answer
用 source document answer
Step 3: Answer Comparison
一致 → correct
不一致 → hallucination
Input / Output
Input
document
summary
Output
QA consistency score
