## 当前系统边界

虽然主流程已经跑通，但仍需明确当前版本的边界：

- 真实 embedding 已接入，但当前仍保留 API 失败后的 hash fallback
- Qdrant 后端已是批处理默认主后端，但 API/配置层默认值仍保留 local 以兼容现有调用
- 图像检索默认已切到原生 CLIP 视觉向量；若本地视觉模型不可用，仍会回退到代理文本路径
- reranker 已同时支持文本专用重排与图像候选的多模态 VL 重排
- analysis artifact 已新增 `retrieval_runtime` 字段，用于明确当前是否启用原生多模态路径；当前默认已切到 `clip + qwen_vl`
- 区域级证据、局部对齐、客户印象建模还未进入主链路

因此当前版本可以定义为：

- 一个已跑通的 MVP 闭环
- 一个具备工程扩展位的检索与分析系统
- 但还不是最终研究版或生产版召回质量
