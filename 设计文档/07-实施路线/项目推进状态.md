## 当前系统边界

虽然主流程已经跑通，但仍需明确当前版本的边界：

- 真实 embedding 已接入，但当前仍保留 API 失败后的 hash fallback
- Qdrant 后端已经可用，但尚未切换为默认主后端
- 图像检索当前仍基于图像代理文本，而不是真实视觉向量
- reranker 已接入专用文本重排模型，但当前仍是基于文本内容重排，尚未进入真正视觉 cross-encoder 阶段
- 区域级证据、局部对齐、客户印象建模还未进入主链路

因此当前版本可以定义为：

- 一个已跑通的 MVP 闭环
- 一个具备工程扩展位的检索与分析系统
- 但还不是最终研究版或生产版召回质量
