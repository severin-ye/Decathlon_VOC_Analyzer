# 设计文档总览

本目录用于把 [../gpt聊天记录.md](../gpt聊天记录.md) 中的项目设计讨论整理为可维护、可扩展、单一职责的文档系统。

## 文档边界

- 本目录关注系统设计、技术路线、数据建模、召回设计、生成逻辑、可解释性与实施顺序。
- 本目录不保存原始对话全文，原始来源仍以 [../gpt聊天记录.md](../gpt聊天记录.md) 为准。
- 每篇文档只负责一个主题，避免把背景、实现、选型、评估混在同一文件中。
