# 引用目录总览

本目录用于沉淀原始对话中出现过的外链资料，供后续论文写作、方案答辩和方法章节撰写使用。

## 整理原则

每条引用都尽量按统一结构记录：

- 外链
- 对应内容整理
- 本项目使用的部分
- 使用的原因
- 本项目使用的思路

## 使用建议

如果你后续写论文：

- 方法背景可优先引用基础综述、原始模态召回、结构化召回三组资料。
- 系统实现部分可优先引用 LangChain、Qwen、Qdrant、Docling。
- 相关工作对比部分可直接使用“候选但未采用路线”中的 OpenClaw、ClawRAG 等材料。

## 未纳入的外链

以下外链没有纳入正式引用目录：

- 对话本身的 ChatGPT 分享链接
- ChatGPT Exporter 页面

原因是它们不是项目技术依据，也不适合作为论文引用来源。
