# 商品证据模型

## 商品证据包的组成

每个商品以 product_id 为核心，挂载以下证据：

- 基础信息：标题、品类、价格、品牌、SKU
- 商品文案：短描述、长描述、卖点、参数、规格
- 商品图片：主图、细节图、场景图、结构图、尺码图

## 核心原则

商品图和商品文字不是两套独立系统，而是同一个商品证据包的两个模态视图。

## 最低字段建议

### 商品文本块

- text_block_id
- product_id
- text_type
- content
- source_section

### 商品图片

- image_id
- product_id
- image_type
- image_path 或 image_url
- optional aux_text

## aux_text 的作用

即使走 Native Multimodal 路线，也建议为商品图片保留轻量辅助文本：

- 图片标题
- 简短描述
- 页面位置
- 可选 OCR 摘要

这不是要替代原始模态，而是为了后续调试、过滤、展示和错误分析更容易。
